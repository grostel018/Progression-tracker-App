# Archive

This directory stores older versions of the project that were removed from the live repository root during cleanup.

The active application still lives at the repository root:

- `config/`
- `DB/`
- `public/`
- `scripts/`
- `src/`

Archived material is kept here so historical iterations remain available without creating multiple live app roots.

Current archive groups:

- `2026-03-repo-cleanup/`
  - `home/`: older root-level static auth prototype
  - `legacy/`: previously archived prototype assets and static versions
  - `Progression tracker app/`: nested duplicate copy of the app
  - `Progression_Tracker_App/`: older pre-`src/` app layout
